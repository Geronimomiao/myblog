# 进程 线程
* 一个进程下可以有多个线程
* 进程中的任意一线程执行出错，都会导致整个进程的崩溃
* 线程之间共享进程数据
* 当一个进程关闭之后，操作系统会回收进程所占用的内存
* 进程之间的内容相互隔离( ICP机制解决进程间通信问题 )
