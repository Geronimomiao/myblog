## 计算单位
****
* 容量单位
	* 在物理层面 高低电平记录信息
	* 理论上只认识 0/1 两种状态( 0/1 称为 bit(比特位

![enter description here](https://img.wsmpage.cn/learning/2019-11-22/1574392553658.png)
	
	
	
* 速度单位
	* 网络速度
		* Mbps
		* 100M/s = 100Mbps = 100Mbit/s = (100/8)MB/s = 12.5MB/s
	* CPU速度
		* Hz( 每秒周期性变动重复次数的计量
		* 1GHZ=10^3MHZ=10^6KHZ= 10^9HZ
